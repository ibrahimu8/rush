# rush

Fast parallel HTTP requests in C.

## Install

git clone https://github.com/ibrahimu8/rush
cd rush
make

## Usage

Single URL
rush https://example.com

Multiple URLs (parallel)
rush https://api1.com https://api2.com https://api3.com

## Why rush?

Fast: True parallel execution
Simple: No dependencies, just works
Safe: Validated and timeout handling built-in

## Requirements

libcurl
